Subform Element Module
------------------------
by Wolfgang Ziegler, nuppla@zites.net

This module just provides a new form element, that can be used by other modules.
If you are a developer, read API.txt.

Installation
------------
 * Copy the subform element directory to your modules directory and activate the
   subform element module.
